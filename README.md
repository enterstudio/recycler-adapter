# Installation

**Gradle**

``` gradle
repositories {
  maven { url "https://jitpack.io" }
}
```

``` gradle
dependencies {
  compile 'com.github.moezbhatti:recycler-adapter:0.1.1'
}
```

**Maven**
``` xml
<repositories>
  	<repository>
      <id>jitpack.io</id>
      <url>https://jitpack.io</url>
	</repository>
</repositories>
```

``` xml
<dependency>
    <groupId>com.github.moezbhatti</groupId>
    <artifactId>recycler-adapter</artifactId>
    <version>0.1.1</version>
</dependency>
```
